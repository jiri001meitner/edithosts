Postup pro instalaci

Nejprve se ujistěte, že máte operační systém Linux (například Ubuntu, Fedoru, OpenSUSE nebo Linux Mint, atd.), máte nainstalovaný Git, jste root nebo máte právo sudo.

cd /tmp
git clone https://github.com/jiri001meitner/edithosts.git
edithosts/install.sh

Edithosts má otevřený zdrojový kód, každý si může snadno ověřit, jestli v něm není chyba. Kdo najde chybu, nebo dostane nápad, jak kód vylepšit, tak může projekt tzv. forknout (zjednodušeně řečeno zkopírovat a vést vlastní vývojovou větev) nebo mne kontaktovat.
